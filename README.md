## Descripción
Máquina virtual básica del lenguaje tipo ensamblador Bies. Interpreta su sintaxis y muestra la salida de su ejecución. Implementado en NodeJS y antlr4

```bash
generate_parser.bat
```

```bash
node index.js
```
